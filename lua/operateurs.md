# Opérateurs

### Mathématiques

* addition :**+**
* soustraction :**-**
* division :**/**
* multiplication :**\***
* modulo :**%**

##### Exemples:

```lua
pods_courant = 100
pods_max = 1000
pods_pourcentage = (100 / 1000) * 100 -- 10
```

```lua
x = 10 % 1 -- 0
y = (10 + 10) % 3 -- 2
```

### De comparaison

* Est égal à : ==
* Est différent de : ~=
* Est supérieur à : &gt;
* Est inférieur à : &lt;
* Est supérieur ou égal à : &gt;=
* Est inférieur ou égal à : &lt;=

### Logiques

* Et: _and_
* Ou: _or_
* Non \(inverser le résultat de l'expression\): _not_
